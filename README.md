# Portfólio Fictício - TechStore

Este é um projeto de portfólio desenvolvido como parte da disciplina de **Desenvolvimento Web**. 
O objetivo foi aplicar conceitos fundamentais de estruturação e estilização para criar uma interface funcional, limpa e responsiva.

---

## Tecnologias Utilizadas

Para a construção deste projeto, foram utilizadas as seguintes tecnologias:

* **HTML5:** Estruturação semântica do conteúdo (tags `header`, `nav`, `main`, `footer`).
* **CSS3:** Estilização visual, layout, tipografia e organização de elementos.
* **Git & GitHub:** Controle de versão e hospedagem através do GitHub Pages.

---

## Estrutura do Projeto

O projeto está organizado da seguinte forma:

* `index.html`: Página principal (Início).
* `produtos.html`: Exibição dos produtos fictícios.
* `contato.html`: Formulário de contato com validação básica.
* `/css`: Pasta contendo os arquivos de estilização (`style.css`).

---

## Aprendizados

Durante o desenvolvimento deste trabalho, pude reforçar:
1.  O uso de **HTML Semântico** para melhor acessibilidade e SEO.
2.  Organização de arquivos e caminhos relativos (links entre páginas e CSS).
3.  Fluxo de trabalho com Git (commit, push e deploy).
